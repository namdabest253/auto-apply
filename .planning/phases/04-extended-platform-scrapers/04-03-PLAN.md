  <action>
Create src/lib/scrapers/handshake.ts:
- `HandshakeScraper` class implementing ScraperAdapter with `platform = "handshake"`
- Constructor takes `userId: string` (needed to look up credentials from UserSetting)
- `discover(params)`:
  1. Load encrypted credentials from UserSetting (key="handshake_credentials") for this.userId
  2. If no credentials, log warning and return empty array
  3. Decrypt credentials using decrypt() + getEncryptionKey()
  4. Call `loginAndSearch(credentials)`

- `loginAndSearch(creds)`:
  1. Create stealth browser via `createStealthBrowser()`
  2. Navigate to `https://app.joinhandshake.com/login`
  3. Enter university name in the school search field, select from dropdown
  4. Click continue -- redirects to university SSO page
  5. Handle common SSO providers:
     - Okta: fill username + password fields, click sign in
     - CAS: fill username + password fields, submit form
     - Generic: look for email/username and password input fields, fill and submit
  6. Wait for redirect back to Handshake (URL contains `app.joinhandshake.com`)
  7. If MFA/Duo prompt detected (iframe or push notification), wait up to 60 seconds for user to complete manually, then continue
  8. Navigate to search: `https://app.joinhandshake.com/postings?category=Internship&page=1`
  9. Use Handshake's native filters: job type = Internship/Co-op, location = United States
  10. Extract jobs from search results page: title, company, location, URL, date
  11. Paginate through results (next page button), cap at 10 pages
  12. Close browser

- Error handling: wrap entire flow in try/finally (always close browser). On SSO timeout or failure, log error and return empty array. Do NOT throw -- the worker should continue with other scrapers.
